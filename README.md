# Raptor OS

A custom Bazzite-based Linux distribution with Brave Browser, Fastfetch, and a neon green KDE theme.

## Installation

Download the latest ISO from the [Releases](https://github.com/cerberus9-dev/Raptor-OS/releases) page, flash it to a USB drive using [Balena Etcher](https://etcher.balena.io) or [Ventoy](https://www.ventoy.net), and boot from it to install.
