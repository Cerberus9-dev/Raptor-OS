# Raptor OS

A custom Bazzite-based Linux distribution with Brave Browser, Fastfetch, and a neon green KDE theme.
(HEAVY W.I.P, FEEDBACK IS APPRECIATED)

## Installation

Download the latest ISO from https://archive.org/details/raptor-os, flash it to a USB drive using [Rufus]https://rufus.ie/en/) or [Ventoy](https://www.ventoy.net), and boot from it to install.
